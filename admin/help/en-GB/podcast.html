Podcast Entry - Help Screen

Overview

The Podcast Entry form allows you to create and edit podcast feeds. Podcasts generate RSS feeds that can be submitted to Apple Podcasts, Spotify, Google Podcasts, and other podcast directories.

Toolbar Actions

Button Description
Save Saves the podcast and remains on the edit form.
Save & Close Saves the podcast and returns to the Podcasts list.
Cancel Discards any unsaved changes and returns to the list.
Help Opens this help screen.

Details Tab

Basic Information

Field Description
Podcast Title (Required) The name of the podcast as it will appear in podcast apps.
Alias URL-friendly version for the feed URL.
Description Description of the podcast shown in directories and apps.
Status Publication status: Published generates the feed, Unpublished disables it.
Language Primary language of the podcast content.

iTunes/Apple Podcasts Settings

Field Description
iTunes Category Primary category for podcast directories (e.g., Religion & Spirituality).
iTunes Subcategory Subcategory within the main category.
iTunes Author Author name shown in podcast apps.
iTunes Owner Name Name of podcast owner for directory contact.
iTunes Owner Email Contact email for podcast directories.
Explicit Content Mark if podcast contains explicit content.

Image Tab

Field Description
Podcast Image Cover art for the podcast. Required for most directories.
Image Requirements: Apple Podcasts requires a square image, minimum 1400x1400 pixels, maximum 3000x3000 pixels, in JPEG or PNG format.

Feed Settings

Field Description
Items per Feed Number of episodes to include in the RSS feed.
Download Link Control how download links appear in the feed.
Website Link URL to your podcast's website or landing page.
Tip: After creating a podcast, assign media files to it from the Media Files edit screen. Only published media files assigned to a podcast will appear in the RSS feed.

Feed URL

Your podcast feed URL will be:
https://yoursite.com/index.php?option=com_proclaim&view=podcast&id=X

Submit this URL to podcast directories like Apple Podcasts and Spotify.
