Templates List - Help Screen

Overview

The Templates list displays all display template configurations. Templates control the visual presentation of Proclaim content on the frontend.

Toolbar Actions

Button Description
New Create a new template.
Actions Dropdown with: Publish, Unpublish, Archive, Check In, Trash, and Batch options.
Help Opens this help screen.

List Columns

Column Description
Checkbox Select items for batch operations.
Status Publication status. Click to toggle.
Template Title Name of the template. Click to edit.
Access Access level.
ID Unique identifier.

Filtering and Search

  • Search - Search by template title
  • Status - Filter by publication status

Batch Operations

Select multiple templates to change access level in bulk.

Note: Template ID 1 is the default template and cannot be unpublished or deleted.
