Message Entry - Help Screen

Overview

The Message Entry form allows you to create and edit sermon/study records. Messages are the core content of Proclaim, representing individual sermons, Bible studies, or teachings. Each message can have multiple media files attached.

Toolbar Actions

Button Description
Save Saves the message and remains on the edit form.
Save & Close Saves the message and returns to the Messages list.
Save & New Saves the current message and opens a blank form for a new one.
Save as Copy Creates a duplicate of the current message.
Cancel Discards any unsaved changes and returns to the Messages list.
Help Opens this help screen.

Details Tab

Basic Information

Field Description
Title (Required) The sermon/study title as it will appear on the website.
Alias URL-friendly version of the title. Auto-generated if left blank.
Study Date The date the message was delivered or recorded.
Teacher The speaker who delivered this message.
Series Optionally assign this message to a series.
Message Type Category of message (e.g., Sermon, Bible Study, Devotional).
Location Where the message was delivered.
Topics Tags/topics associated with this message for filtering and search.

Scripture References

Field Description
Book Primary Bible book referenced.
Chapter/Verse Specific chapter and verse references.
Secondary Reference Additional scripture passages covered.

Content

Field Description
Study Text Full text, notes, or outline of the message. Supports rich text formatting.
Status Publication status: Published, Unpublished, Archived, or Trashed.
Access Viewing access level for this message.
Language Language assignment for multilingual sites.

Media Files Tab

View and manage media files attached to this message. Click "New" to add audio, video, or document files. Each message can have multiple media files.

Tip: Add multiple media types (audio, video, notes PDF) to make your messages accessible in different formats.

Images Tab

Field Description
Message Image Featured image for this message. Displayed in lists and social sharing.
Thumbnail Smaller version for compact displays.

Publishing Tab

Standard Joomla publishing information including created/modified dates and users.

Permissions Tab

Configure access control for this specific message.

Best Practices

  • Use descriptive titles that help visitors find relevant content
  • Always include scripture references for easy searching
  • Assign topics to improve discoverability
  • Add a featured image for social media sharing
  • Include study notes or outlines in the Study Text field
