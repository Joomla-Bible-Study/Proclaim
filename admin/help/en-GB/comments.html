Comments List - Help Screen

Overview

The Comments list displays all user comments submitted on messages. Use this screen to moderate comments - approve, edit, or remove them.

Toolbar Actions

Button Description
New Create a new comment manually.
Actions Dropdown with: Publish, Unpublish, Archive, Check In, Trash, and Batch options.
Help Opens this help screen.

List Columns

Column Description
Checkbox Select items for batch operations.
Status Publication status. Click to toggle (approve/unapprove).
Study The message the comment is attached to.
Full Name Commenter's name.
Comment Date When the comment was submitted.
ID Unique identifier.

Filtering and Search

  • Search - Search comment text
  • Status - Filter by publication status
  • Language - Filter by language

Batch Operations

Select multiple comments to:

  • Change language assignment
  • Change access level
  • Publish or unpublish in bulk
Tip: Filter by "Unpublished" status to see comments awaiting moderation.
