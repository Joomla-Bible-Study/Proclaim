Location Entry - Help Screen

Overview

The Location Entry form allows you to create and edit location records. Locations represent where sermons or studies were delivered, such as church buildings, conference venues, or retreat centers.

Toolbar Actions

Button Description
Save Saves the location and remains on the edit form.
Save & Close Saves the location and returns to the Locations list.
Cancel Discards any unsaved changes and returns to the Locations list.
Help Opens this help screen.

Details Tab

Field Description
Location Name (Required) The name of the location as it will appear on the website.
Alias URL-friendly version of the name. Auto-generated if left blank.
Status Publication status: Published, Unpublished, Archived, or Trashed.
Access The access level required to view this location.
ID The unique database identifier.

Image Tab

Field Description
Location Image Photo of the location. Click Select to choose from the media manager.
Tip: Location images help visitors identify different venues, especially useful for churches with multiple campuses.

Publishing Tab

Standard Joomla publishing information including created/modified dates and users.

Permissions Tab

Configure access control for this specific location record.
