Template Entry - Help Screen

Overview

The Template Entry form allows you to configure display templates. Templates control how messages, media files, and other content are displayed on the frontend. Each template has multiple settings tabs for fine-tuned control.

Toolbar Actions

Button Description
Save Saves the template and remains on the edit form.
Save & Close Saves the template and returns to the Templates list.
Cancel Discards any unsaved changes and returns to the list.
Help Opens this help screen.

Details Tab

Field Description
Template Title (Required) A descriptive name for this template configuration.
Status Publication status.
Access Access level required to use this template.

List Display Settings

Controls how message lists are displayed:

Field Description
Show Title Display the message title in lists.
Show Teacher Display the teacher/speaker name.
Show Date Display the study date.
Show Scripture Display scripture references.
Show Series Display the series name.
Show Image Display thumbnail images.
Show Description Display message description/excerpt.

Item Display Settings

Controls how individual message detail pages are displayed:

Field Description
Show Full Text Display the complete message text.
Show Media Files Display attached media files.
Show Topics Display topic tags.
Show Social Share Display social sharing buttons.

Media Display Settings

Controls how media files appear:

Field Description
Media Layout How media items are arranged (list, grid, etc.).
Show Duration Display media duration.
Show File Size Display file size information.
Tip: Create multiple templates for different display needs - one for the main sermon archive, another for a sidebar widget, etc.
Note: Template ID 1 is the default template and cannot be unpublished or deleted.
