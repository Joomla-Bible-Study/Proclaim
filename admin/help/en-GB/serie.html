Series Entry - Help Screen

Overview

The Series Entry form allows you to create and edit sermon/study series. A series groups related messages together, such as a multi-week teaching series or a book study.

Toolbar Actions

Button Description
Save Saves the series and remains on the edit form.
Save & Close Saves the series and returns to the Series list.
Save & New Saves the current series and opens a blank form for a new one.
Cancel Discards any unsaved changes and returns to the Series list.
Help Opens this help screen.

Details Tab

Field Description
Series Title (Required) The name of the series as it will appear on the website.
Alias URL-friendly version of the title. Auto-generated if left blank.
Teacher Default teacher for this series. Individual messages can override this.
Description Description of the series content displayed on series detail pages.
Status Publication status: Published, Unpublished, Archived, or Trashed.
Access The access level required to view this series.
Language Language assignment for multilingual sites.
ID The unique database identifier.

Image Tab

Field Description
Series Image Main image representing the series. Displayed on series listings and detail pages.
Thumbnail Smaller version for use in compact lists.
Tip: Create compelling series images to attract visitors. Recommended size: 800x450 pixels (16:9 ratio).

Publishing Tab

Standard Joomla publishing information including created/modified dates and users.

Permissions Tab

Configure access control for this specific series record.
