Locations List - Help Screen

Overview

The Locations list displays all location records in the system. Locations indicate where sermons or studies were delivered.

Toolbar Actions

Button Description
New Create a new location.
Actions Dropdown with: Publish, Unpublish, Archive, Check In, Trash, and Batch options.
Help Opens this help screen.

List Columns

Column Description
Checkbox Select items for batch operations.
Status Publication status. Click to toggle.
Location Name Name of the location. Click to edit.
Access Viewing access level.
ID Unique identifier.

Filtering and Search

  • Search - Search by location name
  • Status - Filter by publication status
  • Access - Filter by access level

Batch Operations

Select multiple locations to change access level in bulk.
