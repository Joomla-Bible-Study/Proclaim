Topics List - Help Screen

Overview

The Topics list displays all topic tags in the system. Topics help categorize messages by subject matter, making content more discoverable.

Toolbar Actions

Button Description
New Create a new topic.
Actions Dropdown with: Publish, Unpublish, Archive, Check In, and Trash options.
Help Opens this help screen.

List Columns

Column Description
Checkbox Select items for batch operations.
Status Publication status. Click to toggle.
Topic Name The topic text. Click to edit.
ID Unique identifier.

Filtering and Search

  • Search - Search by topic name
  • Status - Filter by publication status
