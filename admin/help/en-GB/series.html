Series List - Help Screen

Overview

The Series list displays all sermon/study series in the system. Series help organize related messages into groups for easier navigation and discovery.

Toolbar Actions

Button Description
New Create a new series.
Actions Dropdown with: Publish, Unpublish, Archive, Check In, Trash, and Batch options.
Help Opens this help screen.

List Columns

Column Description
Checkbox Select items for batch operations.
Status Publication status. Click to toggle.
Series Title Name of the series. Click to edit.
Access Viewing access level.
Language Language assignment.
ID Unique identifier.

Filtering and Search

  • Search - Search by series title
  • Status - Filter by publication status
  • Access - Filter by access level
  • Language - Filter by language

Batch Operations

Select multiple series to change language or access level in bulk.

Tip: Use series to organize topical studies, book studies, or seasonal sermon series.
