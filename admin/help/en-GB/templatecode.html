Template Code Manager - Help Screen

Overview

Template Codes allow advanced users to create custom PHP template files that replace the default Proclaim views. This feature provides complete control over how sermons, series, teachers, and other content are displayed on your website.

Once created, a template code file can be selected in your Template Display Settings to override the default layout for a specific view type.

Advanced Feature: Template Codes require knowledge of PHP, HTML, and CSS. Incorrect code can break your site's display. Always test changes on a development site first.

Creating/Editing Template Codes

Form Fields

Field Description
Filename Enter a name for your template file (without the .php extension). Use lowercase letters, numbers, and underscores only.
Type Select the view type this template will replace. Cannot be changed after creation.
Template Code The PHP/HTML/CSS code for your custom template. Uses CodeMirror editor with syntax highlighting.
Published Set to Published to make the template available for selection in Template Display Settings.

Template Types

Type Description
Sermon List Replaces the default sermon/study listing page showing multiple sermons.
Sermon Replaces the single sermon/study detail view.
Teachers Replaces the teacher listing page showing all teachers.
Teacher Replaces the single teacher detail view.
Series Displays Replaces the series listing page showing all series.
Series Display Replaces the single series detail view with its studies.
Module Custom template for the Proclaim module.
Tip: When you create a new template code, default starter code is automatically inserted. Do not modify the initial PHP code at the top of the file. Add your custom code at the bottom.

Best Practices

  • Always back up your template codes before making changes
  • Test your template code on a development site before deploying to production
  • Use meaningful filenames that describe the template's purpose
  • Add PHP comments to document your custom code
  • Do not modify the default code that appears at the top of new templates
  • Check for PHP errors in Joomla's error logs if your template doesn't display correctly
