Template Codes List - Help Screen

Overview

The Template Codes list displays all template code definitions. Template codes are reusable code snippets that define how individual elements are rendered within templates.

Toolbar Actions

Button Description
New Create a new template code.
Actions Dropdown with: Publish, Unpublish, Archive, Check In, and Trash options.
Help Opens this help screen.

List Columns

Column Description
Checkbox Select items for batch operations.
Status Publication status. Click to toggle.
Code Name Name of the template code. Click to edit.
Type The type of element this code renders.
ID Unique identifier.

Filtering and Search

  • Search - Search by code name
  • Status - Filter by publication status
  • Type - Filter by code type
Advanced: Template codes require knowledge of PHP and Joomla's templating system. Modifying these incorrectly can break your site's display.
