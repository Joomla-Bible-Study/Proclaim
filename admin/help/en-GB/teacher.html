Teacher Entry - Help Screen

Overview

The Teacher Entry form allows you to create and edit teacher/speaker records. Teachers are associated with messages (sermons/studies) and can have biographical information, contact details, and images.

Toolbar Actions

Button Description
Save Saves the teacher and remains on the edit form.
Save & Close Saves the teacher and returns to the Teachers list.
Cancel Discards any unsaved changes and returns to the Teachers list.
Help Opens this help screen.

Details Tab

Basic Information

Field Description
Teacher Name (Required) The full name of the teacher as it will appear on the website.
Alias URL-friendly version of the name. Auto-generated from the name if left blank.
Title Optional title or honorific (e.g., "Pastor", "Rev.", "Dr.").
Status Publication status: Published, Unpublished, Archived, or Trashed.
Access The access level required to view this teacher (Public, Registered, etc.).
Language Language assignment for multilingual sites. Use "All" for all languages.
ID The unique database identifier. Auto-generated for new records.

Contact Information

Field Description
Website URL to the teacher's personal website or ministry page.
Email Contact email address for the teacher.
Phone Contact phone number.
Address Physical or mailing address.

Biography

Field Description
Short Biography A brief summary of the teacher displayed in lists and compact views.
Full Biography Complete biographical information displayed on the teacher's detail page.

Image Tab

Field Description
Teacher Image Photo or avatar image for the teacher. Click Select to choose from the media manager.
Thumbnail Smaller version of the image used in lists. Can be auto-generated or uploaded separately.
Tip: For best results, use square images (e.g., 400x400 pixels) for teacher photos.

Publishing Tab

Standard Joomla publishing information:

Field Description
Created Date and time the record was created.
Created By User who created the record.
Modified Date and time of last modification.
Modified By User who last modified the record.

Permissions Tab

Configure access control for this specific teacher record. Set permissions for different user groups to control who can edit, delete, or change the state.
