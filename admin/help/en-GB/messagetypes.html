Message Types List - Help Screen

Overview

The Message Types list displays all message type categories in the system. Message types help classify and filter content by the type of teaching.

Toolbar Actions

Button Description
New Create a new message type.
Actions Dropdown with: Publish, Unpublish, Archive, Check In, Trash, and Batch options.
Help Opens this help screen.

List Columns

Column Description
Checkbox Select items for batch operations.
Status Publication status. Click to toggle.
Message Type The type name. Click to edit.
Access Viewing access level.
ID Unique identifier.

Filtering and Search

  • Search - Search by type name
  • Status - Filter by publication status
  • Access - Filter by access level

Batch Operations

Select multiple types to change access level in bulk.
