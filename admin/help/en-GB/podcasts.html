Podcasts List - Help Screen

Overview

The Podcasts list displays all podcast feeds configured in the system. Each podcast generates an RSS feed that can be submitted to podcast directories.

Toolbar Actions

Button Description
New Create a new podcast feed.
Actions Dropdown with: Publish, Unpublish, Archive, Check In, Trash, and Batch options.
Help Opens this help screen.

List Columns

Column Description
Checkbox Select items for batch operations.
Status Publication status. Click to toggle.
Podcast Title Name of the podcast. Click to edit.
Language Primary language of the podcast.
ID Unique identifier.

Filtering and Search

  • Search - Search by podcast title
  • Status - Filter by publication status
  • Language - Filter by language

Batch Operations

Select multiple podcasts to:

  • Change language assignment
  • Change access level
  • Change download link settings
Tip: You can create multiple podcasts to separate content by topic, speaker, or media type.
