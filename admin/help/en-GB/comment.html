Comment Entry - Help Screen

Overview

The Comment Entry form allows you to view and edit user comments on messages. Comments can be moderated, edited, or removed from this screen.

Toolbar Actions

Button Description
Save Saves the comment and remains on the edit form.
Save & Close Saves the comment and returns to the Comments list.
Cancel Discards any unsaved changes and returns to the list.
Help Opens this help screen.

Details Tab

Field Description
Study The message this comment is attached to.
Full Name Name of the person who submitted the comment.
Email Email address of the commenter.
Comment Text The actual comment content. Can be edited for moderation.
Status Publication status: Published (visible), Unpublished (hidden), or Trashed.
Language Language assignment.
Comment Date When the comment was submitted.
Tip: Use the status field to moderate comments. Set to Unpublished to hide inappropriate comments without deleting them.
