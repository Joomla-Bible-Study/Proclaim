Servers List - Help Screen

Overview

The Servers list displays all media server configurations. Each server defines a source for media files such as local storage, YouTube, Vimeo, or cloud storage services.

Toolbar Actions

Button Description
New Create a new server configuration.
Actions Dropdown with: Publish, Unpublish, Archive, Check In, Trash, and Batch options.
Help Opens this help screen.

List Columns

Column Description
Checkbox Select items for batch operations.
Status Publication status. Click to toggle.
Server Name Name of the server. Click to edit.
Server Type The type of server (Local, YouTube, etc.).
ID Unique identifier.

Filtering and Search

  • Search - Search by server name
  • Status - Filter by publication status

Batch Operations

Select multiple servers to change access level in bulk.
