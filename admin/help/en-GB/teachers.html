Teachers List - Help Screen

Overview

The Teachers list displays all teacher/speaker records in the system. From here you can create new teachers, edit existing ones, publish/unpublish, and perform batch operations.

Toolbar Actions

Button Description
New Create a new teacher record.
Actions Dropdown menu with: Publish, Unpublish, Archive, Check In, Trash, and Batch options.
Help Opens this help screen.

List Columns

Column Description
Checkbox Select items for batch operations.
Status Publication status icon. Click to toggle published/unpublished.
Teacher Name Name of the teacher. Click to edit.
Access The viewing access level for this teacher.
Language Language assignment (shown on multilingual sites).
ID Unique database identifier.

Filtering and Search

Use the filter bar to narrow down the list:

  • Search - Search by teacher name
  • Status - Filter by publication status
  • Access - Filter by access level
  • Language - Filter by language (multilingual sites)

Batch Operations

Select multiple teachers and use the Batch button to:

  • Change the language assignment
  • Change the access level
Tip: Click on column headers to sort the list by that column.
