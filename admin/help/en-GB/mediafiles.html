Media Files Manager - Help Screen

Overview

The Media Files Manager allows you to view, organize, and manage all media files associated with your sermons and Bible studies. Media files can include audio recordings, video files, PDF documents, and other downloadable resources that accompany your messages.

Column Descriptions

Column Description
Checkbox Select media files to perform bulk actions. Click the checkbox in the header to select all visible items.
Ordering Drag the vertical ellipsis icon to reorder media files within the same study. Ordering must be selected as the sort column for drag-and-drop to work.
Published Shows the publication status. Click the icon to toggle between Published, Unpublished, Archived, or Trashed states.
Resource Name The server type and media resource identifier. Click to edit the media file record.
Study Title The title of the sermon or study this media file is attached to.
Media Type The type of server or service hosting the media (e.g., Local, YouTube, Vimeo, Amazon S3).
Create Date The date when the media file record was created.
Access The access level required to view this media file (e.g., Public, Registered, Special).
Stats (Plays/Downloads) Shows usage statistics: number of times played and number of downloads.
ID The unique database identifier for the media file record.

Toolbar Actions

Button Description
New Create a new media file record. You will be taken to the media file edit form.
Actions Dropdown menu with status change options:
  • Publish - Make selected media files visible on the frontend
  • Unpublish - Hide selected media files from the frontend
  • Archive - Move selected media files to archived status
  • Trash - Move selected media files to trash
  • Batch - Open batch processing dialog for bulk changes
Empty Trash Permanently delete all trashed media files. Only appears when viewing trashed items.
Help Opens this help screen.

Filtering and Search

Use the filter bar to narrow down the list of media files:

Filter Description
Search Search by study title or alias. Type id:123 to search for a specific media file ID.
Status Filter by publication status: Published, Unpublished, Archived, Trashed, or All.
Year Filter media files by the year they were created.
Access Filter by access level (Public, Registered, etc.).
Language Filter by language assignment. Only visible when multilingual mode is enabled.

Sorting

Click any sortable column header to sort the list. Available sort options include:

  • Study Title (ascending/descending)
  • Create Date (ascending/descending)
  • Published Status (ascending/descending)
  • Access Level (ascending/descending)
  • ID (ascending/descending)
  • Ordering (for manual drag-and-drop arrangement)
Tip: To use drag-and-drop ordering, select "Ordering" as your sort column. Media files are grouped by study, so you can only reorder files within the same study.

Batch Processing

The batch processing feature allows you to apply changes to multiple media files at once. Select the media files you want to modify, then click Actions > Batch.

Available batch options:

  • Language - Assign a language to selected media files (multilingual sites only)
  • Access Level - Change the access level for selected media files
  • Player - Set the media player to use for playback
  • Popup - Configure popup behavior for media playback
  • Media Type - Change the media type classification
  • Link Type - Set how the media link behaves (download, play inline, etc.)

Check-in / Check-out

When a user edits a media file, it becomes "checked out" to prevent conflicts. If you see a lock icon next to a media file, it means another user is currently editing it.

Administrators can check in locked items by selecting them and using the check-in feature, or by accessing the Global Check-in from the System menu.

Understanding Media File Types

Media files can be hosted on various servers and services:

  • Local/Legacy - Files stored on your web server
  • YouTube - Embedded YouTube videos
  • Vimeo - Embedded Vimeo videos
  • Amazon S3 - Files stored in Amazon S3 buckets
  • Direct Link - External URLs to media files
Note: Each server type may have different configuration options. The Resource Name column shows the server type and the primary identifier for the media resource.
