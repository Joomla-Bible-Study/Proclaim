Administration - Help Screen

Overview

The Administration screen provides access to system-level settings and maintenance tools for Proclaim. From here you can configure component options, manage database tables, and perform maintenance tasks.

Toolbar Actions

Button Description
Options Open the component options/configuration screen.
Help Opens this help screen.

Administration Tabs

Database

Tools for managing the Proclaim database:

  • Check Tables - Verify database table integrity
  • Fix Tables - Repair common database issues
  • Reset Hits - Reset view counters on messages

Backup

Create and restore backups of your Proclaim data:

  • Export - Create a backup file of all Proclaim data
  • Import - Restore data from a backup file
Important: Always create a full site backup before performing database operations or restores.

Archive

Manage archived content:

  • View archived messages and media files
  • Restore archived items to published status
  • Permanently delete archived content

Assets

Manage and rebuild the Joomla assets table for Proclaim:

  • Check Assets - Verify asset permissions are correct
  • Fix Assets - Rebuild asset records
When to use: Run asset fixes if you experience permission errors or if items don't appear when they should.

Component Options

The Options button opens the component configuration with these sections:

General Settings

  • Default template selection
  • Items per page
  • Date format options
  • Default images

Permissions

  • Configure access control for component features
  • Set permissions by user group

Integration

  • Social sharing options
  • Analytics integration
  • Third-party service connections
Tip: Review the component options after installation to configure Proclaim for your specific needs.
